# FREERTOS快速入门

## 前言

**C：** 本系列介绍 FreeRTOS 在 MCU 上的典型用法：任务、队列、信号量、软件定时器与低功耗等，帮助你在资源受限环境中组织多任务软件架构。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 任务创建、调度与优先级设计
- 队列、信号量、互斥量与事件组
- 软件定时器与通知
- 内存与栈：常见踩坑与调试建议
- 与具体芯片/IDE 的移植与裁剪要点（随文注明）

## 学习建议

- 多任务问题优先用调试打印 + 逻辑分析仪/示波器核对时序，避免纯猜。
- 关注栈深度与中断嵌套配置，这是现场最容易出诡异问题的地方。
