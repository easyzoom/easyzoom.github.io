# 数据库

## 前言

**C：** 本册聚焦在"**真实项目里和数据库打交道的那部分手感**"——SQL 写法、索引设计、执行计划、迁移与故障排查。偏实战与经验，而非关系代数证明。具体到某一数据库的产品化细节，请移步同分组下的「MySQL」等专题册。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- SQL 日常技巧与常用函数速查
- 查询优化：索引、执行计划、统计信息
- 事务、锁与并发：隔离级别、MVCC、常见死锁成因
- 数据建模：范式 / 反范式取舍、宽表、分库分表
- 迁移与演进：DDL 在线化、版本管理、数据回填
- 运维视角：监控指标、慢查询、备份与恢复
- NoSQL 视角补充：Redis、ES、时序库的选型边界

## 学习建议

- 任何"优化"都要以**当前执行计划 + 当前数据量**为准，别背结论。
- 本地用小数据复现问题时留意统计信息刷新，否则常会"看起来优化了但上线无效"。
- 和业务 owner 对齐："这条查询该多快、多慢不能忍"比任何调参都重要。
