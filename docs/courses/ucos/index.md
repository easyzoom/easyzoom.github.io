# UCOS快速入门

## 前言

**C：** 本系列介绍 µC/OS（Micrium）类经典 RTOS 在 MCU 上的基本概念：任务、信号量、消息队列与时间片，并配合移植与裁剪要点（具体以所用版本与授权为准）。

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 内核初始化与任务创建
- 同步与通信原语
- 中断与临界区
- 移植层（OS_CPU）关注点
- 与商业授权、版本差异相关的说明

## 学习建议

- µC/OS-II 与 III、开源版本与商业支持路径不同，学习时请先确认 **版本与许可证**。
- 结合官方书籍与当前 BSP 的移植示例对照阅读。
