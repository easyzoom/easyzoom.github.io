# RK系列实践

## 前言

**C：** 本系列整理基于瑞芯微（Rockchip）SoC 的开发实践，包括 U-Boot/内核、BSP、多媒体与常见开发板上的系统定制与驱动适配笔记。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 获取与编译 SDK/BSP 的一般流程
- 设备树、分区与启动介质
- 内核与驱动适配思路
- 图形与多媒体栈（视平台与版本）
- 量产烧录与升级相关注意点

## 学习建议

- RK 芯片型号与 SDK 分支绑定紧密，阅读时请核对 **芯片与仓库分支**。
- 官方文档与社区补丁更新频繁，以当前 SDK Release Note 为准。
