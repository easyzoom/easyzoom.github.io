# RUST语言学习

## 前言

**C：** 本系列从 Rust 所有权与借用出发，介绍模式匹配、错误处理、trait、并发与异步生态入门，并穿插与 C/FFI、嵌入式（`no_std`）的衔接点。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 所有权、生命周期与借用检查器
- `enum`、`Result`/`Option` 与错误处理习惯
- trait 与泛型
- 模块系统、`cargo` 与测试
- 并发与异步（`async`）入门

## 学习建议

- 编译器错误信息是重要学习材料，建议逐条读懂再改，而不是盲目复制修复。
- `edition` 与工具链版本会影响语法与标准库，注意 `rustup` 与 `Cargo.toml` 配置。
