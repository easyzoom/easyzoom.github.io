# Linux内核开发

## 前言

**C：** 本系列面向具备一定 C 与操作系统基础的读者，介绍 Linux 内核构建、模块开发、常见子系统（如进程调度、内存、文件系统、网络栈）的入门路径与调试手段。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 获取源码、配置与编译内核
- 内核模块与字符设备入门
- 同步、锁与并发基础
- 内核调试与诊断：SysRq、printk、ftrace、kgdb、kdump 等
- 阅读源码与向社区贡献的注意事项

## 学习建议

- 内核版本与发行版配置差异很大，请始终以你本机 `uname -r` 与文档标注为准。
- 修改内核前务必备份与可在虚拟机中复现，避免影响工作机稳定性。
