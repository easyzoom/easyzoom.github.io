# 开源库的世界

## 前言

**C：** 本系列挑选日常工程中常用的开源库与中间件，从选型、集成、许可证注意到升级与漏洞治理，偏「带案例的读书笔记」而非百科全书式罗列。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 如何评估库的活跃度、社区与许可证风险
- 包管理器与依赖锁定（语言生态各异）
- 与业务代码的边界：封装、版本升级策略
- 安全公告（CVE）与替代方案思路

## 学习建议

- 引入新依赖前至少确认许可证、维护状态与二进制体积/性能代价。
- 具体库的 API 以官方文档为准，本册只记录个人使用场景与结论。
