# C++语言学习

## 前言

**C：** 本系列在 C 语言基础上，介绍 C++ 的面向对象、模板、RAII、STL 与现代 C++（C++11 及以后）特性，兼顾可读性与工程实践。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 类与对象、继承与多态
- 智能指针、RAII 与异常安全
- 模板与泛型编程入门
- STL 常用容器与算法
- 现代 C++ 特性选讲（lambda、移动语义等）

## 学习建议

- 先巩固 C 的内存与指针概念，再进入类和对象会顺畅很多。
- 编译器与标准版本（`-std=`）会直接影响可用特性，文中示例会尽量注明。
