# MCU开发与实践

## 前言

**C：** 本系列面向单片机应用开发，涵盖外设（GPIO、定时器、UART、ADC 等）、时钟与低功耗、调试手段，以及从裸机到简单 RTOS 的过渡思路。

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 工程结构与启动流程（启动文件、向量表）
- 常用外设编程与硬件抽象习惯
- 调试：SWD/JTAG、日志与示波器配合
- 低功耗与看门狗等可靠性话题
- 与具体厂商 SDK 相关的实践（随文注明芯片型号）
