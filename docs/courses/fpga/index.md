# FPGA学习与实践

## 前言

**C：** 本系列从数字逻辑与硬件描述语言（以 Verilog/VHDL 常见路径为主）出发，介绍 FPGA 开发流程、仿真、时序与板上调试的基本方法，并配合实践笔记。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 组合/时序逻辑与有限状态机
- 仿真与 Testbench 基础
- 综合、实现与时序约束入门
- 常见 IP 与总线接口实践
- 板级调试与信号完整性意识

## 学习建议

- FPGA 与软件编程思维差异大，建议每个知识点都配合仿真或小实验闭环。
- 工具链与器件型号强相关，阅读时注意文中标注的厂商与版本。
