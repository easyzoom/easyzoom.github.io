# AI基础与原理

## 前言

**C：** 本册聚焦在"**理解模型在做什么**"这一层，而不是"怎么调 API"。理解越深，后续做提示工程、做 Agent、做 Vibe Coding 时就越不容易被表象带跑。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- Transformer 架构要点：注意力、位置编码、KV Cache
- Tokenizer、上下文窗口与成本模型
- 采样策略：temperature、top-p、top-k、repetition penalty
- 提示工程的结构化方法（角色、约束、示例、输出模式）
- RAG：检索-重排-生成的最小闭环
- 评测：离线指标、在线 A/B 与人类反馈
- 安全与幻觉：越狱、注入、数据泄露边界

## 学习建议

- 先跑通一次最小 `prompt → 模型 → 输出` 的本地 demo，再谈优化。
- 阅读论文前先看一遍对应的开源实现，代码比公式更直白。
- 模型与接口迭代极快，阅读时请留意文末或 frontmatter 中的时间背景。
