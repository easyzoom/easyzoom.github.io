# Docker的开发与实践

## 前言

**C：** 本系列面向需要在开发、测试与部署中使用容器的读者，覆盖 Docker 基础概念、镜像与容器管理、Compose、网络与存储，以及常见排错思路。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 镜像、容器、仓库与 Dockerfile
- docker compose 编排与多容器协作
- 网络、卷与数据持久化
- 与 CI/CD、私有仓库结合的实践
- 常见问题与日志排查

## 学习建议

- 动手时用非 root 用户组或 rootless 模式时，注意权限与挂载路径差异。
- 生产环境还需结合安全基线、镜像扫描与资源限制，本册以开发与自运维场景为主。
