# Git入门与实践

## 前言

**C：** 本系列面向日常用 Git 协作的开发者，从仓库、提交、分支与合并讲起，再到变基、Cherry-pick、子模块与常见冲突处理，并穿插团队协作习惯。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 工作区、暂存区与提交历史
- 分支策略与合并、rebase 的取舍
- 远程仓库与协作流程（含 PR/MR 习惯）
- `.gitignore`、钩子与部分进阶话题
- 典型问题：冲突、误提交、历史改写的安全做法

## 学习建议

- 涉及历史改写的操作前先确认是否已推送、是否影响他人分支。
- 图形工具与命令行对照使用，有利于理解 Git 对象模型。
