# PYTHON语言学习

## 前言

**C：** 本系列从 Python 语法与标准库入手，逐步涉及虚拟环境、包管理、常用第三方库与简单工程化（测试、类型提示、打包），适合作为脚本与后端入门的线性笔记。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 语法、数据结构与控制流
- 函数式特性、迭代器与生成器
- 模块、包、`venv` / `pip` 基础
- 文件、网络与并发入门
- 与数据、自动化、Web 衔接的选讲

## 学习建议

- Python 2 与 3 已分明，本册默认 **Python 3**，示例会注明最低版本。
- 养成 `requirements.txt` 或现代锁文件习惯，避免「在我机器上能跑」。
