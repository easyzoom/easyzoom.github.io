# C语言学习

## 前言

**C：** 本系列面向希望打好系统编程与嵌入式基础的读者，从 C 语法、指针与内存、预处理器，到与硬件/OS 打交道的常见范式，循序渐进整理成册。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- C 语言核心语法与控制结构
- 指针、数组、字符串与内存模型
- 结构体、联合体与位运算
- 与链接、编译单元相关的基础概念
- 在 Linux/嵌入式场景下的常见实践与踩坑

## 学习建议

- 每章尽量配合可编译、可运行的小例子，避免只记语法不动手。
- 遇到指针与内存问题，可结合调试器与 Valgrind（或目标平台等价工具）验证理解。
