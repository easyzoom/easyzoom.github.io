# ROS2入门与实践

## 前言

**C：** 本系列介绍 ROS 2 的基本概念：工作空间、节点、话题、服务、参数与 launch，以及在仿真或真实机器人上的部署与调试入门。

<!-- more -->

## 本册内容范围

计划围绕下列方向展开（随写作进度持续调整）：

- 发行版选择与安装（Ubuntu 等）
- 工作空间、`colcon` 构建与包结构
- 通信模型：Topic / Service / Action
- 参数、launch 与生命周期
- 与 Gazebo 等仿真、传感器驱动的衔接

## 学习建议

- ROS 2 与 ROS 1 不兼容，命令与工具链也不同，请按 ROS 2 文档学习。
- 版本（Humble、Iron、Jazzy 等）差异明显，示例会尽量注明发行版。
